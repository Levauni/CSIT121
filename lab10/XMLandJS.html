The 411 on XML and JavaScript


What is XML?

XML:
is a text-based syntax designed to describe, deliver, and exchange structured informaton.
  1. How is XML used?
  • Using XML, developers can create any tags they need to describe their information.
  • XML is not intended to replace HTML, but to extend the power of HTML by separating data from presentation.
  • XML is used to store or transport data, while HTML is used to format and display the same data.

What is JavaScript?

JavaScript:
is an object-based, client-side scripting language interpreted by a web browser. JavaScript is considered to be object-based because it’s used to manipulate the objects associated with a web page document: the browser window, the document itself, and elements such as forms, images, and hyperlinks.
  1. How is JavaScript related to HTML and CSS?
  • HTML creates and structures the websites content, CSS adds style and formatting to these HTML structures, and the JavaScript turns those stylized components into something that a user can interact with.
